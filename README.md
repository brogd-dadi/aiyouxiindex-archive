# aiyouxiindex-archive

## 项目简介

`aiyouxiindex-archive` 是一个用于归档和发布独立 HTML 页面的仓库。仓库本身不针对任何特定域名或网站，仅作为页面资源的历史存档与公开存放点。所有页面均为独立 HTML 文件，可直接在浏览器中打开查看。

## 目录说明

```
aiyouxiindex-archive/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面依赖的资源文件（样式、脚本、图片等）
└── README.md       # 本文件
```

- `pages/`：包含所有归档的 HTML 页面，按名称或日期组织。
- `assets/`：存放页面所需的公共资源，部分页面可能引用此目录下的文件。

## 页面归档说明

本仓库收录的 HTML 页面均为独立完整文件，不依赖外部服务或后端接口。页面内容可能包含静态信息、工具界面、展示页面等，具体用途请参考各页面标题与注释。

归档页面不会频繁更新，每次提交代表一个历史版本。如需查看特定时间点的页面状态，可切换至对应 commit 或 tag。

## 使用方式

1. 克隆或下载仓库到本地。
2. 打开 `pages/` 目录下的任意 `.html` 文件，即可在浏览器中查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正现有内容。
- 不接收外部提交的页面归档请求，除非与仓库已有内容相关。
- 若发现页面显示异常或资源缺失，欢迎提交 Issue 反馈。
